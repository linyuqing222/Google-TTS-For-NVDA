Google TTS For NVDA

Developers: Dao Duc Trung and Nguyen Anh Duc.

Google TTS For NVDA uses the Chrome WASM TTS engine through a managed headless Chrome process.

The add-on exposes Chrome voices as an NVDA synthesizer and includes a Tools menu voice manager for downloading or removing voice packages listed in the bundled voices.json catalog.

Settings include voice, rate, rate boost, pitch, and volume. Voice packages are downloaded only from the Google TTS voice manager.

Repeated short phrases are cached in RAM for the current NVDA session only. The cache is volatile and is cleared automatically when NVDA exits or the computer restarts.
